## Para iniciar el proyecto

Clona el repositorio 
```sh
git clone https://github.com/FernandoAltamirano/pruebatecnica.git
```
Instala dependencias
```sh
npm install
```
Inicia el proyecto
```sh
npm run server
```
Compila el proyecto
```sh
npm run build
```
